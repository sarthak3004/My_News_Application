# News App
This is a native android app that allows users to fetch latest news and add save them to bookmarks. The app uses the NewsAPI to retrieve news data.

### Technologies and Libraries Used
* Android Views
* News API
* Retrofit
* Gson Converter
* Kotlin Coroutines
* Dagger-Hilt
* Glide
* Room
